# dsp2b
> 戴森球计划 蓝图社区

## 运行

```bash
npm i
# 复制环境变量, 如果你不开发后端的话, 直接复制 .env.production 即可
cp .env.example .env
# dev
npm run dev
# build
npm run build
```
